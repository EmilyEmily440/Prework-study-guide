## Usage
To use this Prework Study Guide, you can review the notes in each section. For suggestions on what to study first, open the Chrome DevTools by pressing Command+Option+I (macOS) or Control+Shift+I (Windows). A console panel should open either below or to the side of the webpage in the browser. There you will see a list of topics we learned from the prework along with a suggestion on which topic to study first.

This project is used for taking notes and viewing notes throughout the course. There are multiple ways to open the file. 
1. via gitbash navigating to this document using cd, and the 'code .'
2. via a file explorer on my personal laptop
3. via visual studio on my personal laptop
4. via my github page online at github.com when I sign in
OR
 You can easily view the subject code once the file is open by
1. left click the mouse anywhere on the screen
2. select inspect on the dropdown menu under the mouse
3. In the side window select console
4. select which subject you were interested in reading
